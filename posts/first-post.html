How I Built My Portfolio Site (and How You Can Too)

This portfolio is entirely static HTML, CSS, and JS: no build process! It's easy to update, deploys easily on GitHub Pages, and costs nothing to host.

My main tips:

  • Keep your CSS simple: use variables for colors/themes
  • Mobile-first responsive structure: try shrinking your browser window!
  • Use cards and sections to divide content visually
  • Add a dark mode toggle for a modern touch
